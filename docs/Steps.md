# Steps to Build this Project  

## Step 01: Initialize Project  

Create a directory:

```sh
mkdir duplo-pipeline
cd duplo-pipeline
```

Initialize a GIT repo and follow prompts: 

```sh
git init
```

Initialize npm project and follow prompts: 

```sh
npm init
```

## Step 02: Make an API  

Add code to [src](../src) directory.
